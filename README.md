# Analisis-Divergensi-Metode-Iterasi-Titik-Tetap
Dalam project ini dilakukan analisis beberapa bentuk fungsi g(x) untuk melihat perilaku konvergensi dan divergensi iterasi numerik menggunakan Microsoft Excel. Hasil iterasi menunjukkan bahwa salah satu fungsi mengalami divergensi dengan nilai iterasi yang terus membesar hingga menghasilkan error sangat tinggi.
